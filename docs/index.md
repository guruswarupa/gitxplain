<h1 class="main-heading">gitXplain</h1>
<div class="subtitle-under-main">Not a replacement of Git, but a simplified extension of it for boosting productivity.</div>

gitXplain is an AI-powered Node.js CLI that analyzes Git commits, commit ranges, and branch diffs to generate structured, human-readable explanations. Understand what changed, why it changed, and how it impacts your codebase.

## Why gitXplain?

#### Understand Complex Changes
Get detailed explanations of what commits do, why they exist, and how the fixes work. Perfect for code reviews and onboarding.

#### Multiple Analysis Modes
Choose from summary, security audit, code review, line-by-line walkthrough, and more. Get exactly the insights you need.

#### Boost Productivity
Automate commit splitting, release management, and documentation generation. Spend less time writing and more time coding.

#### Flexible & Extensible
Support for 6+ AI providers (OpenAI, Groq, Gemini, etc.). Configure for your workflow with environment variables or config files.

## Key Features

#### Commit Analysis
• Analyze single commits, ranges, or entire branches
• Get summaries or comprehensive analysis
• Line-by-line code walkthroughs

#### Code Review
• AI-powered code review with actionable suggestions
• Security-focused analysis and vulnerability detection
• Issue-oriented analysis for bug identification

#### Commit Management
• AI-assisted commit splitting into atomic commits
• Plan and auto-generate commits for working changes
• Conventional commit message generation

#### Release Management
• Automated release branch merging with version detection
• Automatic release tagging based on version bumps
• Release merge previews and execution
